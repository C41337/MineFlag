MineFlag
========

MineFlag is a Minecraft CTF plugin made for Bukkit! Play with your friends and capture the flag before the opposing team does! Battle through the field and grab that flag and get it to your base to win! MineFlag is based by the server admin for the server admin! Were all about simplicity when coming to making plugins! Simply type "/MF" To bring up the diolag for the plugin! Then type "/MF Help" for information about how to set your arenas for your MineFlag Server! Get MineFlag today and keep the hassle away!
